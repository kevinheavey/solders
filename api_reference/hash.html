class Hash(hash_bytes)

A SHA-256 hash, most commonly used for blockhashes.

Parameters:

hash_bytes (bytes) – the hashed bytes.

LENGTH = 32
static default()

The default Hash object.

Returns:

a Hash object.

Return type:

Hash

Example

>>> from solders.hash import Hash
>>> Hash.default()
Hash(
    11111111111111111111111111111111,
)
